FFmpeg bundled with RogueFetch
================================

RogueFetch bundles unmodified FFmpeg binaries. FFmpeg is executed as a separate
process; none of the FFmpeg libraries are linked into RogueFetch.

Build name        : ffmpeg-n7.1.5-2-g998de74adf-win64-lgpl-shared-7.1
FFmpeg version    : n7.1.5-2-g998de74adf-20260713
Upstream commit   : 998de74adf  (FFmpeg release/7.1 branch)
Licence           : LGPL-3.0-or-later  (LGPL v3 or later)
Linkage           : shared libraries (DLLs), so they can be replaced by the user

This build is configured WITHOUT --enable-gpl. The GPL-only components (libx264,
libx265, libxvid, libxavs2, libdavs2, frei0r, librubberband, libvidstab,
avisynth) are explicitly disabled. It IS configured with --enable-version3, which
makes the applicable licence LGPL version 3.

Replacing the FFmpeg libraries
------------------------------
The FFmpeg libraries are shipped as DLLs alongside ffmpeg.exe. To use your own
build of FFmpeg, replace the av*/sw* DLLs and ffmpeg.exe / ffprobe.exe in this
directory with your own compatible versions. Nothing in RogueFetch prevents
this, and no relinking of RogueFetch is required.
